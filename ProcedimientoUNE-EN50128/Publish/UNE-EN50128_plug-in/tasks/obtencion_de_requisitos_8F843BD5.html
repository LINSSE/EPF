Task: 3.1.1 - Obtención de Requisitos
La tarea de obtención de requerimientos indica las pautas a seguir para obtener, documentar, identificar, clarificar y justificar los mismos.
Disciplines: Requerimientos
Purpose
Realizar la obtención de requerimientos de software del sistema asegurando la calidad de los mismos.
Relationships
RolesPrimary Performer:
  • Gestor de requisitos (RQM)
Additional Performers:
InputsMandatory:
  • Descripción de la Arquitectura del sistema
  • Especificación de Requisitos del Sistema
  • Especificación de Requisitos de Seguridad del Sistema
  • Especificaciones de la Interfaz Externa
  • Plan de Garantía de Calidad del Software
  • Plan de Validación del Software
    		Optional:
    • None
    Outputs
    • Especificación de Requisitos Software (ERS)
