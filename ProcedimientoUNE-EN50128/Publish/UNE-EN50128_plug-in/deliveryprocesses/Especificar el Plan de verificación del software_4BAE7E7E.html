El plan de verificación registrará los diferentes pasos y técnicas a ser utilizados en el proyecto de desarrollo para comprobar el cumplimiento de los requisitos de cada fase. UNE EN 50128 6.2.1.1

Main Description

La especificación del plan de verificación consistirá en un único formulario que ordenará las tareas de comprobación y tendrá como documentos relacionados el documento de ciclo de vida y las diferentes guías con las técnicas de verificación.

Salida: Se deben rellenar las tablas correspondientes en el siguiente formulario: F_PVS_01 Plan de Verificación del Software - Proyecto XX

2- Identificar las tareas de verificación

Se identificarán las tareas de verificación para cada fase, las guías de verificación relacionadas y los roles responsables para el desarrollo de esta tarea.

Estas tareas de verificación tendrán en cuenta los ensayos de caja negra, ensayos de caja blanca, listas de comprobación y otras técnicas de acuerdo con las siguientes guías: A5 Verificación y Ensayos, A6 Integración y A7 Ensayos de Software en Conjunto.
