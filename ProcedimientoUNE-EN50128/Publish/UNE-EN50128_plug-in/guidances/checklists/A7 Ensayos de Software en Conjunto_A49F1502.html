Main Description

A continuación se describen las técnicas y buenas prácticas a tenerse en cuenta en las actividades de ensayos de software de conjunto de los proyectos de desarrollo software. Este conjunto de guías servirá como material de consulta y apoyo referenciado en los pasos y procedimientos del ciclo de vida del proyecto. Asimismo servirá como base para las evaluaciones internas, auditorías y revisiones.
